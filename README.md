# FastMCP Planning Agent

Agentic MCP server for Oracle EPM Planning, based on the FastMCP framework and intelligence patterns from the FCCS Agent.

## Features
- Agentic reasoning for complex Planning queries.
- Integration with Oracle EPM Planning REST API.
- Reinforcement Learning for tool selection optimization.
- Persistent context memory.

## Setup
1. Install dependencies: `pip install -e .`
2. Configure environment variables (see `mcp.json`).
3. Run the server: `python -m cli.fastmcp_stdio`
